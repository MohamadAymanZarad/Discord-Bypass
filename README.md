# DiscordBypass

A powerful, easy-to-use C# WPF application designed to bypass Discord and other VOIP blocking (like Valorant and League of Legends) in regions with Deep Packet Inspection (DPI) or DNS-based restrictions, specifically tailored for users in Egypt.
